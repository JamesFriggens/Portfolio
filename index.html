Wireless Computer Remote

Using a Raspberry Pi Pico and ESP8266, I created a TV remote that allows me to shutdown, restart or lock my computer from WiFi range. The Raspberry Pi Pico lacks WiFi capabilities, so it sends AT commands to the ESP8266 which then opens a socket connection with the specified IP Address. I was inspired by this project when I noticed the lack of documentation for connecting the Raspbery Pi Pico to a WiFi capable device to communicate on its behalf.
